# Slidev Presentation

This folder contains a Slidev deck for comparing AI agents:
Codex (ChatGPT) vs Alfagen (Qwen3 Coder 30B), using branch history.

## Prerequisites

- Node.js 18+
- npm

## Install

```bash
cd docs/presentation
npm install
```

## Run in dev mode

```bash
npm run dev -- --open
```

## Build static output

```bash
npm run build
```

## Export to PDF

```bash
npm run export
```
