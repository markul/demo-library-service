# GetSubscriptionById

## Purpose
Returns one subscription by identifier.

## Endpoint
GET /api/subscriptions/{id}

## Parameters
id (GUID): subscription identifier.

## Examples
- Input: Examples/GetSubscriptionById/Input.md
- Output: Examples/GetSubscriptionById/Output.md

## Responses
- Success: 200 OK
