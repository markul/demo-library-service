# GetSubscriptionTypeById

## Purpose
Returns one subscription type by identifier.

## Endpoint
GET /api/subscriptions/types/{id}

## Parameters
id (GUID): subscription type identifier.

- Output: Examples/GetSubscriptionTypeById/Output.md
