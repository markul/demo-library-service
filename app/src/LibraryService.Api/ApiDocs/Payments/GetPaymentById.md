# GetPaymentById

## Purpose
Returns one payment by identifier.

## Endpoint
GET /api/payments/{id}

## Parameters
id (GUID): payment identifier.

## Examples
- Input: Examples/GetPaymentById/Input.md
- Output: Examples/GetPaymentById/Output.md

## Responses
- Success: 200 OK
- Failure: 404 Not Found
