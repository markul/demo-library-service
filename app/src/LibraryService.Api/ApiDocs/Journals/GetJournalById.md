# GetJournalById

## Purpose
Returns one journal by identifier.

## Endpoint
GET /api/journals/{id}

## Parameters
id (GUID): journal identifier.

## Examples
- Input: Examples/GetJournalById/Input.md
- Output: Examples/GetJournalById/Output.md

## Responses
- Success: 200 OK
- Failure: 404 Not Found
