# CreateJournal

## Purpose
Creates a new journal.

## Endpoint
POST /api/journals

## Parameters
Body: title, issueNumber, publicationYear, publisher.

## Examples
- Input: Examples/CreateJournal/Input.md
- Output: Examples/CreateJournal/Output.md

## Responses
- Success: 201 Created
- Failure: 400 Bad Request
